# Proyecto

## Descripción
Sistema de creación de animaciones con Manim para contenido publicitario de Jercol Technologies.

## Objetivo
Crear animaciones visualmente impactantes que comuniquen propuestas de valor de automatización (WhatsApp, IA, etc.) para redes sociales y campañas Ads.

Quiero que lleves una bitacora del proyecto que sirva como documentacion tanto para revisiones futuras mias como para colaboracion. 

Documenta todo lo que encuentres al pasartelo y vas añadiendo de manera organizada todo lo que vayamos actualizando, los desafios tecnicos que se nos presentaron, la tecnica y logica que utilizamos para superarlos. 

todo ordenado y categorizado, que sea limpio y que se entienda perfectmente. 

## Enfoque creativo
- Animaciones dinámicas y llamativas
- Pensado para retención en redes
- Mensajes claros orientados a conversión
- Estética tecnológica moderna

## Convenciones
- Todas las escenas deben usar branding.py
- Priorizar reutilización de componentes visuales
- Código limpio y modular
